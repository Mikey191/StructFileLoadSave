
		/*Режимы открытия файлов
Режим									описание												Обработка данных начинается с 
	r \ Открытие текстового файла только для чтения. \начала файла
	w \ Открытие текствового файла только для записи. Если такой файл не существует, то он будет создан \ начала файла
	a \ Открытие текстового файла для добавления. Если такой файл не существует, то он будет создан\ Конца файла
	r+ \ Открытие текст файла для чтения и записи без возмодности изменить размер файла \ начала файла
	w+ \ Открытие текст файла для чтения и записи. Если такой файл не сущ, то он будет создан \ начала файла
	a+ \ Открытие текст файла для чтнения и записи.Если такой файл не сущ, то он будет создан\ Конец файла
	rb \ Открытие двоичного файла для чтения. Если такой файл не существует то булет ошибка\начало файла
	wb \ Открытие двоичного файла для записи.Если такой файл не сущ, то он будет создан\Начала файла
	b \ Открытие двоичного файла для добавления. Если такой файл не сущ то он будет создан\ конца файла
	r+b или b+\Открытие двоичного файла для чтения и записи без возможности изменения размера файла\ началоа файла
	w+b или wb+\Открытие двоичного файла для чтения и записи. Если такой файл не существует то он будет создан\начала файла
	a+b или ab+\Открытие двоичного файла для чтения и записи. Если такого файла не существует то он будет создан\конца файла*/
